E-commerce Sales Analysis

Objective
Analyze an e-commerce furniture dataset to identify factors affecting sales, pricing, and customer behavior.

Dataset
- E-commerce Furniture Dataset 2024


Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Power BI
- SQL


Steps Performed
- Data Cleaning (handled missing values, removed duplicates)
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Visualization
- Dashboard Creation


Key Insights
- Products with higher discounts tend to generate more sales
- Higher-rated products show better conversion rates
- Certain categories dominate overall revenue
